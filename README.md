# Event Landing Page

This project is a simple event landing page for a Sandeep Maheshwari seminar, built using HTML and CSS. It provides information about the event, the speaker, schedule, venue, and includes a contact form and social media links.

## HTML Features Used

- **Semantic HTML5 tags:** `header`, `nav`, `main`, `section`, `article`, `footer`, etc. for better structure and SEO.
- **Navigation bar:** Using `<nav>` and `<ul>` for navigation links.
- **Forms:** Using `<form>`, `<input>`, `<textarea>`, `<button>` for user input (ticket purchase and contact).
- **Media integration:** Embedded YouTube video using `<iframe>`.
- **Google Maps integration:** Embedded Google Maps using `<iframe>` for venue location.
- **External CSS:** Linked using `<link>` tag for styling.

## CSS Features Used

- **Internal and External CSS:** Both are used for styling different components.
- **Flexbox:** Used extensively for layout, particularly in the navigation bar, gallery, speaker information, etc.
- **Background images:** Used in the header and other sections to enhance visual appeal.
- **Hover effects:** Applied to buttons, social media icons, and gallery images for interactivity.
- **Media Queries:** Implemented for responsiveness across different screen sizes.
- **Custom fonts and colors:** Used throughout the website to maintain a consistent brand identity.

## Functionality

1. **Homepage (index.html):** 
   - Provides an overview of the event and speaker.
   - Includes sections on the speaker, schedule, and contact information.
   - Features a gallery and social media links.
2. **Venue Page (venue.html):**
   - Detailed information about the event venue.
   - Embedded Google Maps for easy location finding.
   - Additional details like parking information.
3. **Speaker Page (speaker.html):**
   - Biography of Sandeep Maheshwari.
   - Embedded YouTube videos of past talks.
   - Showcase of similar speakers.
4. **Schedule Page (schedule.html):**
   - Breakdown of the event schedule with timings.
   - Information about dates, times, and activities.
5. **Contact Page (contact.html):**
   - Contact form for inquiries.
   - Social media links for different platforms.

## How to Use

1. Clone the repository.
2. Open `index.html` in your web browser.
3. Navigate through different pages using the navigation bar.

## Future Enhancements

- **Ticket Purchase Functionality:** Integrate a payment gateway for actual ticket purchases.
- **Dynamic Content:** Fetch event data, speaker information, and schedule from an external API or database.
- **User Authentication:** Allow users to create accounts to save their ticket purchases and preferences.
- **Improved Accessibility:** Enhance the website's accessibility for users with disabilities.
